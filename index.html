QTpapers by lindenle

A tool for scientists to keep their papers organized in a digital format

Dependencies

qt4

Install

qmake make ./qt_papers

Authors

Loren A. Linden Levy (lindenle@gmail.com)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/lindenle/QTpapers

get the source code on GitHub : lindenle/QTpapers
